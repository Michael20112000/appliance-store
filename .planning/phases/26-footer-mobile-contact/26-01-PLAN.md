| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-26-01 | Denial of Service | `submitCallbackRequestAction` / `callback-request.service` | mitigate | DB count rate limit: 5 requests/hour/IP (D-14); return RATE_LIMIT inline message |

| T-26-04 | Spoofing | IP from `x-forwarded-for` | accept | Trust forwarded headers on Vercel production; dev may mis-attribute — low impact on callback abuse |
| T-26-SC | Tampering | npm installs | mitigate | Phase adds **zero** new packages per RESEARCH Package Legitimacy Audit |
