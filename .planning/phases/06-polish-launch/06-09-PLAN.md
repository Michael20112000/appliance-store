# Plan 06-09: UAT gap — mobile chat scroll

**Mode:** gap_closure  
**Source:** 06-UAT.md test 6  
**Debug:** `.planning/debug/mobile-chat-scroll.md`

## Goal

На mobile (Sheet ≤767px) список повідомлень у чаті скролиться touch-жестом; панель закривається через X у header або sheet close.

## Tasks

### 1. Flex layout у chat-panel

**File:** `src/components/chat/chat-panel.tsx`

- Замінити fragment у `PanelBody` на `<motion.div className="flex min-h-0 flex-1 flex-col overflow-hidden">` (або `div` якщо motion не потрібен).
- На `SheetContent` mobile: додати `min-h-0` поруч з `h-[85dvh] flex flex-col`.
- Опційно `showCloseButton={false}` на mobile Sheet — лишити один X у `PanelHeader`, щоб не дублювати.

### 2. Native scroll на mobile у message-list

**File:** `src/components/chat/message-list.tsx`

- Для списку з повідомленнями: на viewports `<md` рендерити `div` з `className="min-h-0 flex-1 overflow-y-auto overscroll-contain"` замість `ScrollArea`.
- Desktop лишити `ScrollArea` як є.
- Зберегти `bottomRef` + `scrollIntoView` після нових messages.

### 3. Verify

- DevTools iPhone: відкрити чат, 8+ повідомлень, прокрутити вгору/вниз.
- Закрити через X у header.
- `npm test` — без регресій якщо є chat component tests.

## Success criteria

- [ ] Touch scroll працює на mobile sheet
- [ ] Composer залишається внизу, не перекриває тред
- [ ] Close очевидний (одна кнопка X)
