    getAnalyticsData implementation details:
    - sinceDate = new Date(); sinceDate.setDate(sinceDate.getDate() - days)
    - Query 1 ($queryRaw): SELECT DATE("createdAt") AS day, COUNT(*) AS count FROM "Order" WHERE "createdAt" >= ${sinceDate} GROUP BY DATE("createdAt") ORDER BY day ASC
    - Query 2 ($queryRaw): SELECT DATE(o."createdAt") AS day, COALESCE(SUM(oi."priceSnapshot" * oi.quantity), 0) AS revenue FROM "Order" o LEFT JOIN "OrderItem" oi ON oi."orderId" = o.id WHERE o."createdAt" >= ${sinceDate} GROUP BY DATE(o."createdAt") ORDER BY day ASC — NO status filter (per D-01)
    - Query 3: prisma.callbackRequest.count({ where: { createdAt: { gte: sinceDate } } })
    - Day column type guard (per RESEARCH.md Open Question 1): typeof r.day === "string" ? r.day.slice(0, 10) : r.day.toISOString().slice(0, 10)
    - BigInt conversion: Number(r.count) and Number(r.revenue)
    - kpi.totalOrders = ordersByDay.reduce((s, r) => s + r.value, 0) (sum from zero-filled array)
    - kpi.totalRevenue = revenueByDay.reduce((s, r) => s + r.value, 0)
    - kpi.totalCallbacks = callbackCount (plain number from prisma.count)
