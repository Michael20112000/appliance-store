## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Browser → Server Action | Untrusted productId; must pass requireAdmin + service guards |
| Admin session | Only authenticated admins may delete |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-18-01-01 | Spoofing | deleteProductFromListAction | mitigate | requireAdmin() at action entry (same as deleteProductAction) |
| T-18-01-02 | Tampering | productId parameter | mitigate | Reject empty/non-string id with UNKNOWN; deleteProduct service enforces FK/guards |
| T-18-01-03 | Elevation | Non-admin list delete | mitigate | requireAdmin before deleteProduct; no client-only delete path |
| T-18-01-04 | Denial of Service | Accidental mass delete | accept | Single-row confirm per action; no bulk API in scope |
| T-18-01-05 | Information Disclosure | Error toasts | mitigate | Reuse existing UA error codes only; no stack traces in toast |
| T-18-SC | Tampering | npm/pip/cargo installs | accept | No new packages; existing shadcn AlertDialog + sonner |

</threat_model>
