| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-33-07 | Elevation of Privilege | AdminCategoriesTable → reorderCategoriesAction | mitigate | requireAdmin() in the server action (Plan 03) — client-side rendering does not bypass server-side auth |
| T-33-08 | Tampering | handleDragEnd — active.id / over.id are strings from dnd-kit | accept | IDs come from localCategories state which was seeded from server-side data; a tampered ID would cause Prisma RecordNotFound, action returns { ok: false }, UI reverts — no data corruption |
