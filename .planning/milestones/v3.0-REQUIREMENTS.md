# Requirements Archive: v3.0 Chat & Engagement

**Archived:** 2026-05-26
**Milestone:** v3.0 Chat & Engagement (Phases 46–49)
**Original defined:** 2026-05-24
**Core Value:** Покупець швидко знаходить б/у техніку у Львові, оформлює замовлення і за потреби пише магазину в чат.

## v3.0 Requirements (All Complete)

### Guest Access (Гостьовий чат)

- [x] **CHAT-01**: Незареєстрований користувач може відкрити чат-віджет і надсилати текстові повідомлення без реєстрації; сесія зберігається через localStorage (`chat_guest_token`) — **Shipped Phase 46, confirmed by 46-05 UAT**
- [x] **CHAT-02**: Коли гість реєструється або входить в акаунт, його гостьовий чат автоматично прив'язується до акаунту (повідомлення зберігаються) — **Shipped Phase 47**
- [x] **CHAT-03**: У адмінській панелі гостьовий користувач відображається як "Гість" — **Shipped Phase 46, confirmed by 46-05 UAT**

### Lifecycle Control (Управління чатом)

- [x] **CHAT-04**: Адмін може завершити чат; у відкритому віджеті покупця протягом ~1 секунди з'являється "Чат завершено" і input блокується — **Shipped Phase 47**
- [x] **CHAT-05**: Після завершення чату юзер може почати новий (кнопка "Почати новий чат" в банері) — **Shipped Phase 47**

### History Drawer (Історія чатів)

- [x] **CHAT-06**: У заголовку чат-віджету (поряд з кнопкою ×) є кнопка меню, що відкриває дровер з середини віджету; видима тільки для авторизованих — **Shipped Phase 48, confirmed by 48-03 UAT**
- [x] **CHAT-07**: Дровер показує список попередніх чатів з можливістю перемикання між ними — **Shipped Phase 48, confirmed by 48-03 UAT**
- [x] **CHAT-08**: З дровера можна створити новий чат — **Shipped Phase 48, confirmed by 48-03 UAT**

### File Attachments (Вкладення файлів)

- [x] **CHAT-09**: Авторизований юзер та адмін можуть прикріплювати файли (jpg/png/webp, до 10 МБ) у повідомленнях; неавторизовані — тільки текст — **Shipped Phase 49 (PDF dropped, images only)**

| Admin-to-admin internal notes | CRM-фіча |
| Email notification for guests | Гості не мають email |
| Typing indicators (v3.0) | Pusher presence channels; defer |
